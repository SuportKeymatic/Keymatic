  • Line delay (ms)
  • Hotkey (optional global shortcut)

2) Capture a window

Drag the button onto the target window and release.

Tip: you can capture multiple windows in the same button.

3) Create/Edit the script

Right-click the button and select üìù Edit Script.

4) Run

‚ùì FAQ

The script is not working on the target window

  1. Make sure the window was captured correctly
  2. Some programs block simulated keyboard messages
  3. Try running Keymatic as Administrator

Keys are being sent twice

Increase the delay between key sends.

WAIT is not running

  • WAIT runs in parallel with the main loop
  • Use a valid time in ms
  • Close the block with ENDWAIT
  • Avoid nesting WAIT inside REPEAT unless intentional

How do I stop all scripts?

  • Stop each running button
  • Or close Keymatic
Can I use this in games?
Many games prohibit automation in their Terms of Service. Use at your own risk.
